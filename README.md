# Graph API Boilerplate
ES6+ Hapi Server with GraphQL, Neo4j, Babel, ES Lint, and Node Modules: async, lodash, dataloader

## Node Modules
-   Hapi
-   GraphQL
-   Neo4j
-   Babel
-   ESLint
-   Async
-   Lodash
-   Dataloader

## NPM Scripts
`npm test` - runs tests

`npm lint` - runs ESLint

`npm start` - starts the server

## Install
`npm install` - installs dependencies
